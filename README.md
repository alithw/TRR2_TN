# TRR2_TN

Hệ thống ôn thi trắc nghiệm môn Toán Rời Rạc 2 (Đồ thị và Cây).

## Hướng dẫn chạy chương trình

Bạn có thể chạy ứng dụng bằng cách khởi chạy máy chủ Node.js cục bộ:

```bash
node server.js
```

Sau đó, truy cập ứng dụng tại địa chỉ:
[http://localhost:3007](http://localhost:3007)

## Các tính năng chính
- **Ôn tập 305 câu hỏi trắc nghiệm**: Toàn bộ câu hỏi được biên soạn và giải chi tiết.
- **Lời giải chi tiết & Mẹo giải nhanh**: Tách biệt trực quan dạng callout, dễ đọc và tra cứu thuật toán.
- **Thanh điều hướng thông minh**: Duyệt câu hỏi theo thứ tự hiển thị của lưới thay vì nhảy ngẫu nhiên.
- **Giao diện tối ưu hóa**: Thanh sidebar có thể thu gọn thành mini-sidebar `80px` tiện lợi trên desktop.
- **Favicon**: Sử dụng logo môn học làm biểu tượng của website trên trình duyệt.
